## About This Project

Built to highlight my work and technical expertise in web development. Features smooth scroll animations, responsive design, and a clean dark UI.

## Technologies Used

**Frontend**
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- GSAP — scroll-triggered animations
- Lenis — smooth scrolling
